Thank you for using the Process Dashboard!

For help and instructions on the use of the Process Dashboard, you can open the User's Manual at any time by clicking the "" menu on the main window, then choosing "Help → Help Topics."

If this is your first time to run the dashboard, you might benefit from the quick start guide, which provides a quick walkthrough of dashboard usage.

If you will be participating as a member of a Team Project, contact your team leader. They will send you a link you can use to join the team project and create your personal plan.

If you plan to work by yourself (not as a member of a team project), you can create a personal project by clicking the "" menu on the main window, then choosing "New Project."
